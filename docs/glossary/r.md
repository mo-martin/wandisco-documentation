## Replication Rules
File system content is replicated selectively by defining Replication Rules, which specify: the directory in the file system that will be replicated and the Zones that will participate in that replication.

Without any Replication Rules defined, each Zoneâ€™s file system operates independently of the others. With the combination of Zones and Replication Rules, WANdisco Fusion gives you complete control over how data is replicated between your file systems and/or object stores.
